# Change: Initialize Platform Core

## Why
Khởi tạo nền tảng kỹ thuật cơ bản cho hệ thống khóa học sử dụng Astro, MDX và React. Đây là bước nền tảng để các tính năng nội dung và tương tác có thể hoạt động.

## What Changes
- **Astro Scaffolding**: Khởi tạo project với template minimal.
- **Integrations**: Cấu hình `@astrojs/mdx` và `@astrojs/react`.
- **Directory Structure**: Thiết lập các thư mục `src/layouts`, `src/components`, `src/styles`.
- **Global Design**: Thiết lập Typography (Inter/Roboto) và các biến CSS cơ bản.

## Impact
- Specs: `platform-core`
- Code: Khởi tạo `/` project.
